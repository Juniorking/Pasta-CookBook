# 🍝 오늘의 파스타 (Pasta CookBook) — 단일 HTML 버전

파스타 **소스 종류**와 **면 종류**로 골라 보고, 재료·이름·태그로 **검색**할 수 있는 레시피 토이 프로젝트입니다.
이 버전은 **파일 하나(`Pasta-CookBook.html`)로 끝**입니다. 설치도, 서버 실행도 필요 없습니다.

## 실행 방법

`Pasta-CookBook.html` 파일을 더블클릭하면 브라우저에서 바로 열립니다.

- 별도 설치(npm 등) 불필요
- 서버 실행 불필요
- 인터넷은 글꼴을 불러올 때만 사용 — 오프라인이면 시스템 기본 글꼴로 대체되어 동작에는 문제없음

## 기능

- 소스 계열(오일 / 토마토 / 크림 / 페스토)로 필터
- 면 종류(스파게티 / 펜네 / 링귀네 / 페투치네 / 탈리아텔레 / 부카티니 / 트로피에)로 필터 — 면 칩마다 종류별 손그림 아이콘 표시
- 재료·이름·태그 통합 검색 (예: `마늘`, `까르보나라`, `매움`, `해산물`)
- 같은 칩을 다시 누르면 필터 해제(토글)
- 레시피 클릭 시 재료 / 만드는 순서 / 핵심 팁 상세 보기 (모달, `Esc`로 닫기)

## 웹에 올리기 (선택 · GitHub Pages)

서버 없이도 웹에 띄울 수 있습니다.

1. `Pasta-CookBook.html`을 `index.html`로 이름을 바꿔 저장소에 올린다.
2. 저장소 **Settings → Pages**에서 브랜치를 `main`으로 지정하고 저장.
3. 잠시 뒤 생성되는 주소로 접속.

## 레시피 추가·수정하기

`Pasta-CookBook.html`을 텍스트 편집기로 열어 `const RECIPES = [ ... ]` 배열에 객체를 추가하면 됩니다.
소스·면 목록은 데이터에서 자동 추출되므로, 새 종류를 넣어도 필터 칩에 알아서 반영됩니다.

```js
{
  id: "my-recipe",          // 고유 식별자 (겹치지 않게)
  name: "표시 이름",
  nameIt: "Italian Name",
  type: "오일",              // 오일 / 토마토 / 크림 / 페스토
  noodle: "스파게티",
  difficulty: "쉬움",        // 쉬움 / 보통 / 어려움
  time: 15,                  // 분
  servings: 2,
  summary: "한 줄 소개",
  ingredients: [{ name: "재료", amount: "분량" }],
  steps: ["순서 1", "순서 2"],
  tip: "핵심 팁",
  tags: ["태그1", "태그2"]   // 검색에 함께 걸림
}
```
